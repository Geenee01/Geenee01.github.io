Sequence Alignment and Differential Gene Expression Analysis

Bioinformatics Tools : Sequence Alignment, Data Management and Linux Command Line, High-Performance Computing , Cluster Computing, Bioinformatics Tools and Pipelines, RNA-Seq Data Processing, Genome Indexing and Alignment, Data Quality Control, Genomic Data Visualization, Differential Gene Analysis, Gene Expression Quantification,Scripting and Automation.

This project involves a comprehensive workflow for analyzing RNAseq data, combining sequence alignment, transcript assembly, and differential gene expression analysis. The primary objective is to identify and interpret differentially expressed genes from RNAseq datasets. Sequence Alignment: Utilized Fastqc, HISAT2, StringTie, Ballgown, Samtools for aligning RNA sequences to the reference genome, ensuring high accuracy in mapping reads. Transcript Assembly: Leveraged StringTie for reconstructing transcripts and quantifying gene expression levels, providing insights into gene structure and isoforms. Differential Expression Analysis: Applied the Tuxedo suite (including tools like Cuffdiff) to compare gene expression levels across different conditions, identifying genes with significant expression changes.
